# Modern Kart Tasarımı

Bu proje, HTML ve CSS kullanılarak hazırlanmış modern kart tasarımı örneğidir. Çalışmada üç adet kurs kartı tasarlanmış, arka planda `linear-gradient` kullanılmış ve kart yapıları responsive olacak şekilde düzenlenmiştir.

## Proje İçeriği

- 3 karttan oluşan modern bir arayüz tasarımı
- External CSS kullanımı
- Google Fonts entegrasyonu
- Global CSS reset
- Responsive yapı

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Google Fonts

## Dosyalar

- `index.html`: Ana kart tasarımının HTML dosyası
- `style.css`: Ana kart tasarımının stil dosyası


## Projeyi Çalıştırma

1. Proje klasörünü bilgisayarınıza indirin.
2. `index.html` dosyasını tarayıcıda açın.
