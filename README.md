# Modern Kart Tasarımı

Bu proje, HTML ve CSS kullanılarak hazırlanmış modern kart tasarımı örneğidir. Çalışmada üç adet kurs kartı tasarlanmış, arka planda `linear-gradient` kullanılmış ve kart yapıları responsive olacak şekilde düzenlenmiştir.

Proje içinde ana tasarımın yanında, farklı bir görsel yorum içeren ikinci bir alternatif çalışma da bulunmaktadır.

## Proje İçeriği

- 3 karttan oluşan modern bir arayüz tasarımı
- External CSS kullanımı
- Google Fonts entegrasyonu
- Global CSS reset
- Responsive yapı
- Alternatif kart tasarımı için ayrı klasör

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Google Fonts

## Klasör Yapısı

```text
odev-4-Veyse1-main/
|-- index.html
|-- style.css
|-- css-logo.png
|-- js-logo.jpg
|-- ux-logo.jpg
|-- alternatif-kart/
|   |-- index.html
|   |-- style.css
```

## Dosyalar

- `index.html`: Ana kart tasarımının HTML dosyası
- `style.css`: Ana kart tasarımının stil dosyası
- `alternatif-kart/index.html`: Alternatif kart tasarımının HTML dosyası
- `alternatif-kart/style.css`: Alternatif kart tasarımının stil dosyası

## Projeyi Çalıştırma

1. Proje klasörünü bilgisayarınıza indirin.
2. `index.html` dosyasını tarayıcıda açın.
3. Alternatif tasarımı görmek için `alternatif-kart/index.html` dosyasını açın.
